## ALT 품질 개선

### 변경 대상

- GS자이 천연 일반석고보드
  - 원지 표면과 석고 코어 구조를 보여주는 인포그래픽으로 구체화
- 프리미엄 단열재 PF보드
  - 면재와 페놀수지 발포층 구조를 보여주는 인포그래픽으로 구체화
- 단열재 아이소핑크
  - 독립기포 구조와 압출법(XPS) 단열재 단면을 보여주는 인포그래픽으로 구체화

### 원칙

- 이미지에 표현되는 구조 정보만 ALT에 사용
- 프롬프트·이미지 URL·HTML 구조·JSON-LD 구조는 변경하지 않음
- 상품별 ALT 한 문장 유지

---

## 라왕 각재 정보 정리

- D열의 옵션별 규격 문자열을 파싱하여 인포그래픽 옵션 카드에 우선 표시
  - 라왕다루끼: 30×30×3600mm
  - 후지: 10×40×2100mm
  - 심재: 30×40×2100mm
- HTML 규격표는 옵션별 원문을 반복하지 않고 `상품 옵션별 상이`로 간결하게 표시
- FAQ의 성능 비교 질문을 제거하고, 후지와 심재의 규격·형태 구분 질문으로 교체
- 각재 인포그래픽의 원산지·출처·등급·활용 정보는 제외하고 재단 여부와 길이 확인 위치만 유지

## QA 결과

- 라왕 FAQ와 FAQ JSON-LD 동기화 경로 유지
- 다른 각재 비교 Bullet 미적용 확인
- 이미지 생성 및 Vision 검증은 수행하지 않음
