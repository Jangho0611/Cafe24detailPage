## 오늘 작업 마무리 기록

### 이미지 캐시 버전 관리

- 화면용 인포그래픽 URL에 `v` 버전 파라미터 적용
- Google Sheet 이름 지정 범위 `IMAGE_VERSION` 값을 우선 사용
- 설정값이 없거나 읽기에 실패하면 프로젝트 기본 버전 사용
- 이미지 파일명, O열 저장값, Product Schema 이미지는 기존 방식 유지

### 내수합판 전용 인포그래픽

- 일반합판 비교 Type A의 3단을 `내수합판의 차이` 전용 구성으로 분리
- KS 근거가 있는 내수 접착 성능과 사용 환경 차이 중심으로 안내
- 완전 방수, 외부 사용 및 구조 강도 우위 단정 금지

### UV 코팅 자작합판 전용 인포그래픽

- UV 상도·하도를 표면 등급이 아닌 공정 단계로 분리
- 원판 → 샌딩 → UV 하도 → UV 상도 공정 흐름 적용
- 하도는 후속 마감용 바탕 단계, 상도는 최종 표면 마감 단계로 안내

### 제품별 특화 FAQ

- 실제 Google Sheet 상품을 기준으로 제품 전용 FAQ 우선 적용
- 제품 고유 특징, 실제 비교 대상, 사용·선택 기준의 서로 다른 역할로 구성
- HTML FAQ와 FAQPage Schema가 동일한 `faqItems` 원천을 사용하도록 유지

### 오징어합판 FAQ

- 오징어합판·아로합판 전용 FAQ 3개 적용
- 대표 용도, 곡면 시공이 가능한 구조, 일반합판과의 차이를 안내
- 상품과 무관한 MDF 비교 FAQ 제외

### GCS보드·이보드 인포그래픽 개선

- 이보드는 PP 중공 구조판과 연한 분홍색 XPS 단열재가 결합된 완제품으로 표현
- GCS보드는 회색 Glass Fiber Cement Sheet와 크림색 PIR 단열 심재가 결합된 완제품으로 표현
- 1단 완제품 외관 비교, 2단 구성 요소 역할, 3단 선택 포인트로 역할 분리
- 폭발도, 분해도, 접착층 및 실제 제조 단면 추측 금지
- 두 제품의 구성 요소와 색상·재질 혼합 금지
