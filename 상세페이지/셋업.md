

## 4. 실행 방법

### 방법 A. CMD에서 실행

프로젝트 루트에서:

```bash
node gas/run-images.js
```

또는 gas 폴더 안에서:

```bash
cd gas
node run-images.js
```

---

### 방법 B. 더블클릭 실행

프로젝트 루트에 `run-images.bat` 파일을 만든다.

```bat
@echo off
cd /d "%~dp0gas"
node run-images.js
pause
```

이후에는 `run-images.bat` 더블클릭으로 실행한다.

## 5. 실제 사용 흐름

```text
1. 구글시트에서 S열 체크
2. Apps Script가 T열에 이미지 프롬프트 생성
3. U열 상태가 PROMPT_CREATED가 됨
4. CMD 또는 run-images.bat 실행
5. Node.js가 PROMPT_CREATED 행을 찾음
6. OpenAI API로 이미지 생성
7. 생성된 PNG를 Cafe24 이미지 서버에 업로드
8. O열에 이미지 URL 기록
9. U열 상태를 IMAGE_UPLOADED로 변경
10. O열 URL을 기준으로 P열 상세 HTML 생성
```

---

## 6. 상태값 기준

| 상태 | 의미 |
|---|---|
| READY | 준비 상태 |
| PROMPT_CREATED | T열 프롬프트 생성 완료 |
| GENERATING | 이미지 생성 중 |
| IMAGE_UPLOADED | 이미지 업로드 및 O열 URL 입력 완료 |
| HTML_CREATED | P열 상세 HTML 생성 완료 |
| ERROR | 오류 발생 |

## 8. 오류 시 확인

### `OPENAI_API_KEY` 오류
`.env`에 키가 정확히 들어갔는지 확인한다.

### Google Sheets 접근 오류
`service-account.json`이 `gas/` 폴더에 있는지 확인한다.  
구글시트가 서비스 계정 이메일에 공유되어 있는지도 확인한다.

### Cafe24 업로드 오류
FTP 주소, 아이디, 비밀번호, 업로드 경로를 확인한다.

### 실행은 됐는데 처리할 행이 없음
구글시트 U열 상태가 `PROMPT_CREATED`인지 확인한다.
