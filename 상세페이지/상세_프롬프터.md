# 카페24 상세페이지 HTML 생성 프롬프터
> 작성일: 2026-06-26
> 최종수정: 2026-06-29 (MDF 테스트 피드백 반영)
> 적용 대상: 대산우드랜드 카페24 상품 상세설명 (`{$product_detail}`)
> 참조 문서: 상세_design.md
> **이 프롬프터를 그대로 에이전시에 전달하여 HTML을 생성하라.**

---

## 사용 방법

1. 아래 프롬프터에서 `{열}` 부분을 구글시트 해당 열 값으로 교체
2. 에이전시에 전달
3. 생성된 HTML을 P열에 저장
4. 카페24 상품 상세설명 HTML 모드에 붙여넣기

---

## 프롬프터

```
[규칙: 상세_design.md 전체 준수]
- <!DOCTYPE html> 금지 / 외부 CSS·JS 금지
- 모든 클래스 ds- prefix
- 이미지 https:// URL만 / 모바일 반응형 필수
- 이미지 주소(O열 등)는 사용자가 제공한 원본 URL을 절대 임의로 변경하지 말 것 (GitHub raw URL 등으로 대체 금지)

[폰트]
- 'Pretendard', 'Apple SD Gothic Neo', 'Noto Sans KR', sans-serif

[UI 색상: 상세_design.md 9번 기준 — 그 외 색상 절대 금지]
- 기본배경: #FFFFFF
- 섹션배경: #F8F8F8
- 메인(딥그린): #123628
- 포인트(골드): #C9A84C
- 텍스트: #1C1C1C
- 서브텍스트: #616161
- 보더: #E0E0E0
- 반전텍스트: #FFFFFF

[타입 스케일: 상세_design.md 10번 기준 — 크롬 실측값]
- H2: 24px / 700 (모바일 20px)
- 섹션소제목 (.ds-section-title): 14px / 600 (모바일 13px)
- 스펙표 th: 14px / 600 (모바일 13px)
- 스펙표 td: 15px / 400 (모바일 13px)
- 정의문·본문카드: 15px / 400 (모바일 13px)
- CTA버튼: 15px / 600 (모바일 14px)
- 마무리: 13px / 400
- line-height: 1.75 전체 공통
- max-width: 790px / margin: 0 auto

[제품 데이터]
- 제품명: {B열}
- 규격: {D열} / 두께: {C열}
- 등급: {E열} / 제조사: {F열}
- 핵심표현: {H열} (출처: {I열})
- 강조포인트: {K열}
- 용도: {L열}, {M열}
- 인포그래픽 이미지 URL: {O열}

[페이지 구성 - 순서 고정]

1. H2 헤드라인
   - 형식: {B열} {D열} / {C열 두께범위} / {E열 등급}
   - 예: "MDF 1220×2440 / 3T~30T / E1·E0 등급"
   - 키워드 나열 금지 / 문장형 금지 / 최대 40자
   - 색상: #1C1C1C / font-weight: 700

2. 제품 정의문 (1줄)
   - B열(제품명), I열(출처), H열(핵심표현)을 결합하여 자연스러운 1줄 문장으로 구성 (기존 고정 패턴 폐기).
   - 예시 (MDF): "MDF는 목재 섬유를 고온·고압으로 성형한 중밀도 섬유판으로, KS F 3200 기준 밀도 600~800kg/m³ 범위의 판상 자재입니다."
   - 배경: #F8F8F8 / 좌측 보더: 3px solid #123628
   - 색상: #616161

3. 스펙표
   - 항목: 규격 / 두께옵션 / 등급 / 제조사 / 즉시출고
   - th: 배경 #F8F8F8 / 색상 #616161 / 14px
   - td: 배경 #FFFFFF / 색상 #1C1C1C / 15px
   - 보더: 1px solid #E0E0E0
   - 즉시출고: "재고보유 즉시출고 가능" 고정
   - 모바일 반응형: 화면 폭 768px 이하에서는 th와 td가 모두 display: block; width: 100%; 처리되어 세로로 쌓이도록 작성.

4. 인포그래픽 섹션 제목
   - 스펙표 아래, 이미지 바로 위에 <div class="ds-section-title">구조와 밀도 비교</div> 와 같이 섹션 제목을 추가.
   - 제품군 및 인포그래픽 타입(A/B/C)의 주제에 맞게 타이틀을 적절히 조정.

5. 인포그래픽
   - <img src="{O열}" alt="{B열} 구조와 밀도 비교 인포그래픽"
     style="max-width:100%;height:auto;display:block;">
   - O열 비어있으면 영역 전체 생략
   - **중요**: 제공된 이미지 URL은 절대 변경하지 말 것.

6. 이 제품이 필요한 이유 (3문장)
   - AI 느낌 금지 / 영어 라벨 금지 / 현장 B2B 언어 사용.
   - 배경: #F8F8F8 / 상단 보더: 2px solid #123628
   - **중복 금지**: 인포그래픽 이미지 내부에 표기되는 단면 구조, 조직 균일성, 휨강도 등의 특징은 본문에서 절대 반복 서술하지 말 것.
   - 구성: 후가공성(도장/필름/래핑), 반복 재단 안정성, 재고보유 및 대량 납품 출고 상담 정보 중심.

7. CTA 버튼
   - 배경: #123628 / 글자: #FFFFFF
   - width: 100% / padding: 16px / border-radius: 4px
   - 텍스트: "대량구매 견적 · 규격 확인하기 →"
   - href: https://web-cadalog-ver10.vercel.app/
   - hover 효과: opacity: 0.92만 사용 (새로운 색상 도입 금지)
   - 하단: 전화 031-388-3833 · 평일 09:00–18:00
   - 전화번호 색상: #616161 / 13px / 중앙정렬

8. 마무리
   - 배경: #123628 / 글자: #FFFFFF
   - 중앙정렬 / padding: 20px / 13px
   - "(주)대산 · 35년 신뢰의 건축자재 전문 공급사"
```

## 체크리스트 (생성 후 반드시 확인)

- [ ] `<!DOCTYPE html>` 없음
- [ ] 외부 CSS 파일 링크 없음
- [ ] 외부 JS 라이브러리 없음
- [ ] 모든 클래스 `ds-` prefix
- [ ] `max-width: 790px` 적용
- [ ] 모바일 반응형 포함 (스펙표 block 처리 포함)
- [ ] **제공된 이미지 URL 변경 없이 그대로 사용**
- [ ] **이미지 그래픽 정보와 본문 3문장 내용 중복 배제**
- [ ] `position: fixed` 없음
- [ ] `<form>`, `<iframe>` 없음
- [ ] 색상 상세_design.md 9번 기준 외 없음
- [ ] CTA hover 시 `opacity: 0.92`만 사용
- [ ] 타입스케일 10번 기준 준수
- [ ] H2 형식 준수 (키워드 나열 금지)
- [ ] 즉시출고 문구 포함
- [ ] CTA href 정확히 입력
- [ ] AI 느낌 없음 / 영어 라벨 없음
- [ ] 인포그래픽 바로 위에 `.ds-section-title` 타이틀 존재
