# 인포_타입C.md

## 0. 문서 목적

이 문서는 **카페24 상품 상세설명 영역에 삽입할 C타입 인포그래픽 이미지 생성 기준**이다.

C타입은 **질감·표면특성형 인포그래픽**이다.  
수치적 스펙이나 내부 단면 구조보다 제품의 실제 표면 질감, 색감, 무늬, 마감 상태, 그리고 시각적인 표면 분위기가 구매 결정에 결정적 영향을 미치는 마감재 제품군에 적용한다.

---

## 1. 적용 제품 예시

표면의 시각적·촉각적 질감이 핵심 소구점인 건축자재 및 인테리어 마감 보드류에 적용한다.
- 낙엽송 합판 엠보 (브러싱 가공 표면)
- 오크 미장합판 (천연 무늬목 표면)
- 백색 코팅 합판 (LPM/HPM 등 코팅 표면)
- UV코팅 자작합판 (투명 코팅 광택 표면)
- 인테리어 마감 보드 및 템바보드류
- 기타 표면의 무늬와 질감이 구매 판단의 최우선 기준이 되는 모든 제품

---

## 2. 상세페이지 HTML과 역할 분리

정보의 중복을 피하고 사용자 경험을 최적화하기 위해 상세페이지 HTML 텍스트 영역과 인포그래픽 이미지의 역할을 엄격히 분리한다.

### 2.1 HTML 텍스트가 담당하는 정보
- 제품명 (텍스트 검색 및 SEO 목적)
- 규격 (가로 × 세로 mm)
- 두께 (T 옵션)
- 등급 (친환경 등급 등)
- 제조사 / 수입원
- 용도 및 시공 가이드
- 즉시출고 안내 및 CTA 링크

### 2.2 인포그래픽 이미지가 담당하는 정보
- 표면 질감 (클로즈업 비주얼)
- 색감 및 실제 톤앤매너
- 나뭇결, 패턴 및 무늬의 선명도
- 마감 상태 (광택, 매트, LPM 코팅 등)
- 일반 표면과의 차이 비교 (표면 패턴 비교)
- 핵심 질감 키워드 (표면 특성을 설명하는 짧은 단어)

---

## 3. 사용 데이터 (구글시트 매핑)

이미지 생성 시 구글시트의 특정 데이터 열을 아래 규칙에 맞게 매핑하여 프롬프트를 구성한다.

- **B열 (제품명)**: 2단 표면 비교 영역의 우측 비교 대상 라벨로 딱 1회만 사용한다. (인포그래픽 타이틀이나 설명문 등에 중복 노출 절대 금지)
- **G열 (비교요소)**: 2단 표면 비교 영역의 좌측 비교 대상 라벨로 사용한다. (예: 미송합판)
- **H열 (핵심표현)**: 1단의 질감 표현 및 3단의 키워드 도출을 위한 메인 키워드로 사용한다.
- **J열 (단면구조특징)**: 단면 구조 정보가 있더라도 참고만 하되, 절대 단면 도해를 그리지 않고 오직 **표면 질감** 표현을 풍부하게 만드는 요소로만 해석한다.
- **K열 (강조포인트)**: 원문을 그대로 이미지에 텍스트로 노출하지 않고, 핵심 질감 키워드(단축어)를 도출하는 가이드로만 활용한다.
- **L열/M열 (용도)**: 이미지 내부에 표기하는 것을 **절대 금지**한다. (HTML 텍스트와 중복 방지)

---

## 4. 캔버스 기준

- **사이즈**: 800 × 900 px (가로형 및 세로형의 최적 밸런스 규격)
- **포맷**: PNG
- **배경색**: #FFFFFF (Pure White)
- **용도**: 카페24 상품 상세설명 내 보조 이미지 삽입용
- **구조**: 세로 3단 구성

---

## 5. 색상 기준 (상세_design.md 준수)

상세페이지 브랜드 가이드라인에 정의된 색상 외의 다른 색상 도입을 절대 금지한다.

- **기본 배경**: `#FFFFFF` (Pure White)
- **섹션 배경**: `#F8F8F8` (Light Gray)
- **메인 (강조)**: `#123628` (Deep Green)
- **포인트**: `#C9A84C` (Gold)
- **텍스트**: `#1C1C1C` (Dark Charcoal)
- **서브텍스트**: `#616161` (Medium Gray)
- **보더 라인**: `#E0E0E0` (Light Border)

## 6. Type C 레이아웃 구조 (세로 3단)

```
┌──────────────────────────────────────────────┐
│  [1단] 표면 질감 클로즈업                      │
│  - 나뭇결, 코팅, 광택, 무늬 실사 렌더링       │
│  - 3개 이하의 단순 지시 라벨                 │
├──────────────────────────────────────────────┤
│  [2단] 표면 비교                             │
│  ┌───────────────────┬───────────────────┐   │
│  │   일반 비교대상    │     해당 제품     │   │
│  │   {G열 비교요소}   │    {B열 제품명}   │   │
│  └───────────────────┴───────────────────┘   │
├──────────────────────────────────────────────┤
│  [3단] 핵심 질감 키워드 카드                   │
│  ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐         │
│  │키워드│ │키워드│ │키워드│ │키워드│         │
│  └──────┘ └──────┘ └──────┘ └──────┘         │
└──────────────────────────────────────────────┘
```

### 6.1 1단: 표면 질감 클로즈업 (가장 상단)
- 제품 표면을 줌인(Zoom-in)하여 텍스처를 극대화해 보여준다.
- 가공 상태(엠보싱, 브러싱, 코팅, 옹이 등)가 육안으로 만져질 듯이 생생해야 한다.
- 표면의 특정 부분을 가리키는 지시 라벨은 최대 3개 이하로 제한하며, 짧은 명사형 단어만 사용한다.

### 6.2 2단: 표면 비교 (중앙)
- 일반적인 비교 대상(G열)과 해당 제품(B열)의 표면 상태를 좌우로 대비하여 보여준다.
- 마감감의 차이, 무늬의 선명도 차이 등을 직관적인 실사 그래픽으로 묘사한다.
- 텍스트는 좌우 제품명 라벨 1개씩만 노출한다.

### 6.3 3단: 핵심 질감 키워드 (하단)
- 표면의 특징을 요약하는 3~4개의 카드로 구성한다.
- 긴 문장을 배제하고, 자재 고유의 마감 방식을 설명하는 핵심 구절로 채운다.
- 예: `입체 엠보`, `강한 나뭇결`, `브러싱 질감`, `스테인 효과`

---

## 7. 절대 금지 사항

- **규격, 두께, 등급, 제조사, 용도 표기 절대 금지** (HTML 스펙표와 100% 중복되므로 이미지 내 노출 금지)
- **상세설명 HTML 본문 문장의 반복 사용 금지**
- **강조포인트(K열) 문장의 원문 그대로 노출 금지** (반드시 명사형 키워드로 축약할 것)
- **영어 라벨 및 불필요한 외래어 사용 금지**
- **`SECTION 1`, `S1`, `S2` 등 내부 기획용 코드명 노출 금지**
- **"업계 최저가", "최고의 품질" 등 배너 광고식 과장 광고 문구 금지**
- **이미지 최상단에 대형 제품명을 타이틀로 반복 배치하는 것 금지**

---

## 8. Type C 하이브리드 디자인 원칙

- **실사 텍스처 극대화**: 이미지 생성 모델이 가진 질감 묘사 능력을 최대한 활용하여 실제 목재나 판재 표면과 구분하기 어려울 정도의 고품질 텍스처를 렌더링한다.
- **절제된 타이포그래피**: 그래픽 요소가 메인이 되므로 한글 텍스트는 얇고 깔끔하게 배치하여 B2B 기술 자료의 느낌을 유지한다.
- **감성적 포스터화 지양**: 인테리어 잡지나 감성 포스터처럼 레이아웃을 잡지 말고, 시공업자와 B2B 바이어가 자재의 표면 사양을 분석할 수 있는 단정하고 직관적인 카드 타입 구조를 고수한다.

## 9. ChatGPT API 이미지 생성용 기본 프롬프트

```
건축자재 B2B 쇼핑몰 상세페이지용 한국어 인포그래픽 이미지를 생성하라.

이 이미지는 카페24 상품 상세설명 HTML 안에 삽입되는 보조 이미지다.
광고 포스터가 아니라, 제품의 표면 질감과 마감 특성을 이해시키는 실무형 인포그래픽으로 만든다.

목표:
- 상세설명 HTML 텍스트와 중복 금지 (스펙, 치수, 제조사 기입하지 말 것)
- 표면 질감, 표면 비교, 핵심 질감 키워드만 시각화
- 한눈에 표면감과 마감 차이가 이해되는 B2B 자료 스타일
- 한글 텍스트는 선명하고 정확하게 표시

캔버스:
- 800 x 900px
- 배경 #FFFFFF
- 세로형 3단 구성

색상 (정해진 HSL/Hex 테마 엄수):
- 기본 배경 #FFFFFF
- 섹션 배경 #F8F8F8
- 메인 #123628
- 포인트 #C9A84C
- 텍스트 #1C1C1C
- 서브텍스트 #616161
- 보더 #E0E0E0

1단: 표면 질감 클로즈업
- "{H열 핵심표현}"을 중심으로 제품 표면 질감을 크게 보여준다.
- 나뭇결, 엠보, 코팅감, 광택감, 무늬를 자연스럽게 표현
- 라벨은 3개 이하
- 긴 설명문 금지

2단: 표면 비교
- 좌우 비교 구조
- 왼쪽 라벨: "{G열 비교요소}"
- 오른쪽 라벨: "{B열 제품명}"
- 제품명은 이 비교 라벨에서만 1회 허용
- 비교 차이는 2~3개 짧은 키워드로 표현
- 긴 설명문 금지

3단: 핵심 질감 키워드
- 3개 또는 4개 카드형 키워드 구성
- "{H열 핵심표현}"과 "{K열 강조포인트}"에서 질감 관련 키워드만 추출하여 명사형으로 배치
- 아이콘은 단순하게 표현
- 긴 설명문 금지

절대 금지:
- 규격(mm), 두께(T), 등급(E1 등), 제조사, 용도 표시 금지
- 제품명을 제목이나 설명문으로 반복 금지
- 상세설명 HTML 문장 반복 금지
- 영어 라벨 금지
- 광고 배너 느낌 금지
- 지정 색상 외 추가 금지
```

## 10. 테스트용 프롬프트 (낙엽송 합판 엠보 예시)

건축자재 B2B 쇼핑몰 상세페이지용 한국어 인포그래픽 이미지를 생성하라.

목표:

- 표면 질감, 표면 비교, 핵심 질감 키워드만 시각화
- 한글 텍스트는 선명하고 정확하게 표시

캔버스:
- 800 x 900px

  1. "입체 엠보"
  2. "강한 나뭇결"
  3. "브러싱 질감"
  4. "스테인 효과"
- 텍스트 위에는 심플한 라인 아웃라인 형태의 아이콘 배치.

절대 금지:
- 1220x2440, T, E1, 등급, 규격, 용도 등 텍스트 절대 기입 금지
- 한글 텍스트 깨짐 금지

## 12. 실패 시 수정 기준

*   **표면 질감이 사진에서 잘 느껴지지 않을 때**: 1단 클로즈업 영역의 이미지 스케일을 더 확대(Zoom-in)하고 조명의 대비를 주어 입체감을 살린다.
*   **텍스트가 너무 번잡하게 느껴질 때**: 라벨 및 카드 내부의 서브 설명을 모두 지우고 명사형 키워드(예: `입체 엠보`) 단어만 남긴다.
*   **구조형(Type B)처럼 보일 때**: 자재의 단면 레이어나 내부 도해 요소를 완전히 삭제하고, 카메라 앵글을 오직 자재의 겉표면(Surface)에만 고정한다.
*   **일반 광고용 배너처럼 보일 때**: 배경의 그라데이션, 화려한 이펙트, 과장된 사선 레이아웃 등을 지우고 수평·수직 그리드 기반의 플랫 디자인으로 전환한다.
*   **HTML과 정보가 겹쳐 지루할 때**: 이미지 내부의 용도 설명이나 스펙 단어들을 전부 제거하고 순수 시각 질감 지표로만 채운다.
